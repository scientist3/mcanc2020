          * Error is the fault or mistake in a program

          *Error can occur due to wrong syntax or wrong logic.
          
        - :
            - Many types of errors, classified into following:

            * Syntax/Parse Error:
              - error done by the programmer
              
                < ?php 
                  $x = "North Campus"; 
                  y = "Error in Definig Variable"; 
                  echo $x; 
                  echo $y; 
                ?> 
              

            * Fatal Error:
              - type of error where PHP compiler understand the PHP code but it recognizes an undeclared function
              
                < ?php 
                  function add($x, $y) 
                  { 
                    $sum = $x + $y; 
                    echo "sum = " . $sum; 
                  } 
                  $x = 0; 
                  $y = 20; 
                  add($x, $y); 
                  diff($x, $y);// Error 
                  ?> 
              
            
            * Warning Errors :
              - The main reason of warning errors are including a missing file.
              
                 
              

            * Notice Error: 
              - It is similar to warning error. It means that the program contains something wrong but it allows the execution of script.
